# DropGenius AI V1.0

إمبراطورية الدروب شيبنج الذكية القائمة على Next.js 14.

## 🚀 المميزات
- **نظام أتمتة كامل**: البحث عن المنتجات الرابحة ورفعها بضغطة زر.
- **تسجيل دخول آمن**: مدمج مع Google Identity Services.
- **لوحة تحكم احترافية**: سريعة، متجاوبة، ومصممة بأعلى معايير الـ Glassmorphism.
- **جاهز للنشر**: مهيأ تماماً للعمل على بيئة Netlify.

## 🛠️ التقنيات المستخدمة
- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Auth**: Google One Tap & OAuth

## 📦 التشغيل المحلي
```bash
npm install
npm run dev
```

## 🌐 النشر على Netlify
1. ارفع الكود لمستودع GitHub.
2. اربط المستودع بـ Netlify.
3. تأكد من إضافة المتغيرات البيئية (Environment Variables) في لوحة التحكم الخاصة بـ Netlify.
